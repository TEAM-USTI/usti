# USTI
USTI란 University Student Type Indicator의 약자로 대학생 성격 유형 검사를 뜻합니다.</br>
성신여자대학교 컴퓨터공학과 과동아리 init에서 진행한 팀프로젝트입니다.</br>
http://www.usti.ml 에서 직접 테스트해 볼 수 있습니다.</br></br>
